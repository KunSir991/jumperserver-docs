# 基本设置
## 1 功能简述
!!! tip ""
    - 点击页面左侧菜单的`基本设置`按钮，进入基本设置页面.
    - 可以编辑基本信息，包含当前站点 URL（用户通过外部链接，如邮件跳转到堡垒机的 URL，此处可以填写域名或 IP）、支持自定义全局组织名称。
